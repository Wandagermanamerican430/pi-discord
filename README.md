## 📋 What is pi-discord?

pi-discord is a tool that helps you manage your Discord interactions. It takes mentions, direct messages (DMs), and slash commands and links them to ongoing Pi sessions. This lets you keep track of conversations in one place. The bot runs quietly in the background and handles the routing automatically.

This application works on Windows. It helps you stay organized without needing to switch between chats or apps.

## 🎯 Key Features

- Routes mentions and DMs to active Pi sessions  
- Supports slash commands for easy controls  
- Keeps your conversations persistent over time  
- Lightweight and easy to run on Windows  
- Runs in the background with minimal setup
